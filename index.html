The Holiday Helper

Camera go here.

Instructions

Our project will use emotion-identifying software to detect if a user wants to “kiss” or “miss” a displayed picture of a person. In other words, this will be like swiping on a dating app but with a user’s facial expressions as the input. There will be two panels shown on the screen. One panel will be the webcam and another panel will be a gallery of profile pictures. To swipe left (“miss”), a user will display a disgusted facial expression and to swipe right (“kiss”) a user will display a joyful/smiley facial expression. Once the program detects the facial expression, a status bar will display a user’s response and a new picture will show up on the screen for the user to judge. If we have time we will also display a log of the user’s recorded responses to each profile picture they viewed.
